MI GENTE EXPRESS PORTAL V2

1. In Supabase SQL Editor, run STEP-2-SECURITY.sql.
2. Supabase > Project Settings > API Keys.
3. Copy the PUBLIC publishable key (sb_publishable...).
4. Open config.js and replace PASTE_YOUR_PUBLISHABLE_KEY_HERE.
5. Upload this entire folder as a NEW Netlify project.
6. Log in using the Supabase Authentication user you created.
7. After it works, connect portal.migenteexpress.com to this Netlify project.

Never put the Supabase secret key in config.js.
